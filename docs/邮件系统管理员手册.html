邮件系统管理员手册

访问https://10.20.20.12
在弹出的登录对话框输入邮件系统管理员用户名supervisor和密码进入管理界面
点击“Details”以查看每个项目的详细说明

第一部分更改邮件系统管理员和邮件系统API接口的密码

Email admin项对应访问本界面的管理员密码
Email API user项对应访问邮件系统API接口的密码

第二部分查看或更改邮件系统管理员设置

在页面加载出来时已经填入的是当前系统设定
完成修改后点击“Submit”以应用修改

SMTP server/port和Use SSL选项调整用于发邮件的邮件服务器设置
Sender address是显示在发件人栏的邮箱地址

Receiver address filter rule是用于限制收件人邮箱地址的正则表达式
Email API user只能向符合这个表达式的收件人发送邮件
默认设置只允许用户名全为字母的@cuhk.edu.cn邮箱作为收件人

Test receiver address留空时邮件系统正常工作
当填入邮箱时，所有Email API user发出的都会被重定向到这个邮箱
原本指定的收件人不会收到邮件
此设定用于测试邮件系统

Supervisor address留空时没有影响
当填入邮箱时，Email API user每次成功调用API，该邮箱都会收到一份包含所有目标收件人邮箱地址的邮件
